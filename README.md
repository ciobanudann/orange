## Setup

### To spin the docker container you can use the command below

`make setup`

### If you want to run the application without docker you have to

1. install packages `npm install`
2. npm run dev

## Rerun the application

### To spin the docker container you can use the command below

`make start`

### If you want to run the application without docker you have to

`npm run dev`

### Test

## To run the test

`npm run test`
